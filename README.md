# 🚀 Projeto Hello World - Spring Boot

Este repositório contém um projeto simples desenvolvido em **Spring Boot** como parte das atividades da **Generation Brasil**.

---

## 🛠️ Tecnologias Utilizadas

- Java
- Spring Boot
- Spring Tools Suite (STS)
- Maven
- Git/GitHub

---

## 📂 Estrutura do Projeto

- Controller
- Application
- Maven Dependencies

---

## ▶️ Como Executar

1. Clone este repositório.
2. Importe o projeto no Spring Tools Suite (STS).
3. Aguarde o Maven baixar as dependências.
4. Execute a classe principal da aplicação.
5. Acesse os endpoints pelo navegador ou pelo Postman.
